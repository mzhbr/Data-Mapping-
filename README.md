🗺️ Data Mapping with Cosine Similarity
Automated column mapping between two Excel datasets using cosine similarity — finds the best matching column in file B for each column in file A, based on content.
What It Does
When you have two datasets with differently named but semantically related columns, manual mapping is slow and error-prone. This tool automates it.
Example:
File A has a column: patient_diagnosis
File B has a column: diagnosis_label
The tool maps them automatically based on content similarity
